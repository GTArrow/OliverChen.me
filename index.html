Smart Navigation Application

Mapping applications such as Google Maps have changed the way millions of people live. In this project, I used Google Maps API along with Java for the back end, recreated a more efficient mapping system. The application supports multiple graph search algorithms including breadth-first search, Dijkstra's Algorithm, and A* Search. Users are able to choose any one of them to do navigation.
